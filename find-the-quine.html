You play by generating a random graph. Use "start" and "stop" to see if it looks like a quine. More about chemlambda quines
Think about the graph as if it is alive. Is it? If it does nothing, then is dead. If it grows fast and that's all, then is not interesting. If it grows only a little, then it keeps more or less the same, or it has some parts which grow and some parts which shrink, so there seems to be a balance between growing and shrinking, then you hit the jackpot.
It is OK if the creature lives a while and then it dies. After all, the 10_quine is mortal. 5 new creatures were found, see "new quine?". In order to check they are truly alive, load and start several times, until they live long enough.
Attention! When you hit "load" a "new random graph", each time you get a new one. If you want to reload the same random graph again, then use "reload CODE" button.
If you find something that you think is alive, then copy-paste the "code" of the creature and send a mail to Marius.Buliga@imar.ro. You see the code when you load:
